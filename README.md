# Post-it Notes

A lightweight web application built with Node.js, Express, and EJS that allows users to create, view, and manage colorful digital post-it notes. Each note can have a title, description, and color. Data is stored locally in a JSON file, making the app easy to run, understand, and extend.

## Features

- Create new notes with custom titles and descriptions  
- Choose from multiple background colors for visual organization  
- View all notes in a responsive grid layout  
- Simple and clean user interface  
- Local JSON-based storage (no database required)  
- Easy-to-understand code structure — ideal for learning or prototyping  

## Tech Stack

- **Backend**: Node.js, Express  
- **Template Engine**: EJS  
- **Storage**: JSON file (`notes.json`)  
- **Frontend**: HTML, CSS, vanilla JavaScript
